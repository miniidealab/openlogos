## 技术说明

### 参与方定义

| 别名 | 全称 | 说明 |
|------|------|------|
| U | User | 用户 |
| UI | React UI | 前端界面 |
| Store | Zustand Store | 状态管理 |
| IPC | IPC Bridge | Electron 进程间通信 |
| DB | SQLite Database | 本地数据库 |

### 交互模式

本项目为桌面端纯本地应用，无外部 API 调用。所有数据操作通过以下链路：

```
用户输入 → React UI → Zustand Store → IPC Bridge → SQLite Database
```
