# S03: 管理支出分类 — 测试用例

## 一、单元测试用例

### 1.1 API 字段约束（来源：local-api.yaml → addCategory/updateCategory）

| ID | 描述 | 来源 | 前置条件 | 输入 | 预期输出 |
|----|------|------|---------|------|---------|
| UT-S03-01 | 分类名必填 | addCategory: name required | 无 | `{}` | 400: 缺少必填字段 |
| UT-S03-02 | 分类名长度限制 | categories API | 无 | `{name: "a".repeat(21)}` | 400: 名称超出长度 |
| UT-S03-03 | 分类名唯一性 | categories: name UNIQUE | 已存在"餐饮" | `{name: "餐饮"}` | 400: 分类名已存在 |
| UT-S03-04 | 分类ID必填 | updateCategory: id required | 无 | `{name: "新分类"}` | 400: 缺少必填字段 |
| UT-S03-05 | 分类ID必须存在 | updateCategory | 无 | `{id: 999, name: "test"}` | 400: 分类不存在 |
