# 变更提案：adjust-window-size

## 变更原因

当前窗口默认尺寸为 800x600，最小尺寸为 600x500。作为桌面端记账应用，内容较为丰富（分类标签、金额输入、统计图表等），当前窗口偏小，导致内容显示拥挤，用户体验不佳。

## 变更类型

- [x] 代码级

## 变更范围

- 影响的核心文件：src/main/index.js (BrowserWindow 配置)

## 变更概述

调整 Electron 窗口默认尺寸和最小尺寸：

- **默认尺寸**：800x600 → 1000x700（更宽敞的内容显示空间）
- **最小尺寸**：600x500 → 800x600（避免内容过于拥挤）

修改 main/index.js 中的 BrowserWindow 配置参数。
