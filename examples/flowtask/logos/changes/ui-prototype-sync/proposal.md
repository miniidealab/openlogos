# 变更提案：ui-prototype-sync

## 变更原因
当前实现的页面布局和空间样式与原型图相差较大，需要对齐以确保产品视觉一致性。原型图已在 `logos/resources/prd/2-product-design/2-page-design/` 中定义了完整的设计规范，包括尺寸、间距、颜色和交互细节，但实际代码实现未能忠实还原。

## 变更类型
代码级

## 变更范围
- 影响的需求文档：无
- 影响的功能规格：`2-product-design/1-feature-specs/01-feature-specs.md`
- 影响的业务场景：S01（注册/登录）、S02（任务管理主界面）、S03（分类管理）、S04（账号设置）
- 影响的 API：无（纯前端样式变更）
- 影响的 DB 表：无
- 影响的编排测试：无

## 变更概述
本次变更目标是让三个主要页面的实现与原型图保持一致：

1. **认证页（auth）**：对齐 420px 窗口宽度、Tab 切换样式、表单间距、错误提示位置、按钮样式。

2. **任务主界面（main）**：对齐 860×580px 窗口尺寸、三栏布局比例（左侧边栏 180px、右侧面板 272px）、任务卡片样式、FAB 按钮位置、分类管理弹窗、确认对话框样式。

3. **账号设置页（settings）**：对齐 520px 窗口宽度、用户信息卡片布局、密码修改表单间距、成功提示条样式。

设计系统基准：主色 `#18181B`、强调色 `#2563EB`、成功色 `#16A34A`、危险色 `#DC2626`、圆角 `8px`、字体 Inter、过渡动画 `150ms ease`。
