# RunLogos SVG 标志概念稿

以下为 5 款独立 SVG，可在浏览器中直接打开文件预览。

| 文件 | 思路 |
|------|------|
| `runlogos-concept-a-execute-doc.svg` | 堆叠文档 + 运动轨迹/节点，强调「跑规格」；横版图文标 |
| `runlogos-concept-b-monogram.svg` | 字母 R + 箭头，深色圆角方块，适合 favicon / App 图标 |
| `runlogos-concept-c-play-stack.svg` | 播放三角 + 柱状条（日志/指标感），横版一体字标 |
| `runlogos-concept-d-orbit.svg` | 环形虚线 + 多节点，偏编排/流水线；中心 R |
| `runlogos-concept-e-wordmark.svg` | 极简字标 + 小装饰（绿点 + 速度线） |

配色与 OpenLogos 官网暗色主题协调：**青蓝 `#22d3ee` + 品牌绿 `#0eb677`**，中性灰字重区分 Run / Logos。

若要浅色背景版本，可将 `#fafafa` 改为 `#0f172a`，将灰阶背景形状改为浅灰描边即可。

字体依赖系统 **Inter**；若需「零外部字体」的 SVG，可把 `<text>` 转为 `<path>`（导出轮廓）。
