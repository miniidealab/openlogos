Money-Log - Local API

Local IPC API specification for the Money-Log desktop app (Electron main ↔ renderer).

Version: 1.0.0

Servers

  • ipc://main
    Electron Main Process

Contents

  • Default
    • POST Reset application
    • POST Disable password lock
    • POST Set password
    • POST Verify password
    • POST Add category
    • DELETE Delete category
    • GET List categories
    • PUT Update category
    • GET List expense records
    • POST Save expense record
    • GET Get setting
    • POST Set setting
    • GET Get statistics
