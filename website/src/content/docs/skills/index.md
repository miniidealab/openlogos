OpenLogos includes 12 AI Skills organized by development phase. Each Skill is a Markdown file (`SKILL.md`) that guides AI coding tools through specific tasks.

## Phase 1 — WHY (Requirements)

| Skill | Purpose |
|-------|---------|
| [`project-init`](/skills/project-init) | Guide project initialization |
| [`prd-writer`](/skills/prd-writer) | Write product requirements |

## Phase 2 — WHAT (Product Design)

| Skill | Purpose |
|-------|---------|
| [`product-designer`](/skills/product-designer) | Create product design specs |

## Phase 3 — HOW (Implementation)

| Skill | Purpose |
|-------|---------|
| [`tech-architect`](/skills/tech-architect) | Design technical architecture |
| [`scenario-modeler`](/skills/scenario-modeler) | Model business scenarios |
| [`api-designer`](/skills/api-designer) | Design API specifications |
| [`db-designer`](/skills/db-designer) | Design database schemas |
| [`test-writer`](/skills/test-writer) | Write test cases |
| [`code-generator`](/skills/code-generator) | Generate implementation code |

## Cross-Phase

| Skill | Purpose |
|-------|---------|
