Initialize a new OpenLogos project with the standard directory structure, configuration files, and AI Skills.

## Usage

```bash
openlogos init [project-name]
```

## Arguments

| Argument | Description | Default |
|----------|-------------|---------|
| `project-name` | Name for the new project | Current directory name |

## What it creates

- `logos/logos.config.json` — Project configuration
- `logos/logos-project.yaml` — AI collaboration index
- `logos/resources/` — Document structure for all phases
- `AGENTS.md` — AI tool instruction file
