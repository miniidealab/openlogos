The `openlogos` CLI provides commands for project initialization, synchronization, change management, and test verification.

## Installation

```bash
npm install -g @miniidea/openlogos
```

## Commands

| Command | Description |
|---------|-------------|
| [`init`](/cli/init) | Initialize a new OpenLogos project |
| [`sync`](/cli/sync) | Synchronize AGENTS.md with project configuration |
| [`status`](/cli/status) | Show project phase and progress |
| [`change`](/cli/change) | Create a change proposal |
| [`merge`](/cli/merge) | Merge approved changes |
| [`archive`](/cli/archive) | Archive completed changes |
| [`verify`](/cli/verify) | Verify test results against acceptance criteria |
