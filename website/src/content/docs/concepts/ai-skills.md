title: AI Skills
description: Learn how AI Skills guide coding tools through each development phase.
---

AI Skills are platform-agnostic Markdown files (`SKILL.md`) that provide operational instructions for AI coding tools. Each Skill corresponds to a specific phase or task in the OpenLogos workflow.

Skills are stored in the `skills/` directory and referenced from `logos-project.yaml`. When a developer asks an AI tool to perform a task, the tool reads the relevant Skill and follows its structured instructions.

OpenLogos includes 12 built-in Skills covering the entire development lifecycle.
