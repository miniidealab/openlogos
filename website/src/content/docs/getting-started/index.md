OpenLogos is an open-source software engineering methodology designed for the AI era. It encodes proven software engineering practices as executable AI Skills, enabling developers to use AI coding tools (Cursor, Claude Code, OpenCode, etc.) with a structured, repeatable process.

This section will help you understand what OpenLogos is, how to install it, and how to run your first AI-assisted development session.
